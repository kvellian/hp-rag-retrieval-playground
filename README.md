# ⚡ Harry Potter RAG — Retrieval Playground

This repository contains the code and data artifacts for a Harry Potter themed
Retrieval-Augmented Generation (RAG) system. It includes:

- `src/`: search + retrieval Python modules
- `config/`: retrieval configuration
- `data/`: processed corpus, indexes, embeddings, and evaluation artifacts
- `eda_v2/`: exploratory data analysis reports and plots
- `logs/`: metadata for tier titles

Built for my Data Science Capstone project at DePaul University.
